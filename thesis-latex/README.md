# ParCIO Thesis Template

This directory contains the ParCIO thesis template.
The template can be built either manually (using `pdflatex` or `latexmk`) or using the provided `Makefile`.

## Building via `Makefile`

Before building, ensure that the following required software is installed on your system:
- `pdflatex`
- `latexmk`
- `make`

Optionally, if you want to use SVG figures in your thesis and want to automatically convert them to PDF for LaTeX:
- `inkscape`

To build the template, simply run `make` in the thesis directory.
