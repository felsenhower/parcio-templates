# ParCIO Templates

[![CI](https://github.com/parcio/templates/workflows/CI/badge.svg)](https://github.com/parcio/templates/actions)

This repository contains LaTeX templates for presentations, reports, theses and more.

> [!NOTE]
> There are also Typst packages which can be found in the Typst Universe ([parcio-slides](https://typst.app/universe/package/parcio-slides), [parcio-thesis](https://typst.app/universe/package/parcio-thesis)) and on GitHub ([xkevio/parcio-typst](https://github.com/xkevio/parcio-typst)).

Please see [OVGU's Corporate Design](https://www.cd.ovgu.de/) for more information.
